The MATLAB files were created using MATLAB version R2019b and newer.
Be aware that specific toolboxes are needed to make each file work. Especially the SSVEP_.m files need the psychophysics toolbox, which is quite involved.
Installation can be found here: http://psychtoolbox.org/download
Beginner tutorials can be found here: http://peterscarfe.com/ptbtutorials.html

SSVEP_Single and SSVEP_Quintuple are the files for the signal plate
